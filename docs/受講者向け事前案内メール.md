【準備1】GitHub アカウントの作成

GitHub（ギットハブ）は、世界中のエンジニアが使う
ソフトウェア開発のプラットフォームです。
当日の開発環境として使います。

既にアカウントをお持ちの方は【準備2】へ進んでください。

  1. https://github.com/signup にアクセス
  2. メールアドレスを入力
  3. パスワードを設定
  4. ユーザー名を決める（本名でなくても構いません）
  5. メール認証を完了する

  ※ 無料プランで OK です。クレジットカードは不要です。

【準備2】GitHub にログインできることを確認

  1. https://github.com にアクセス
  2. 右上の「Sign in」からログイン
  3. ログインできれば OK です

【準備3】開発環境の起動テスト（任意・推奨）

当日の環境を事前に試すことができます。

  1. 以下のURLにアクセスしてください。
     （※ 受講者向け URL は当日までに別途お知らせします）

  2. 緑色の「Code」ボタン →「Codespaces」タブ
     →「Create codespace on master」をクリック

  3. 1〜2分でブラウザ上に開発画面（VS Code）が開きます

  4. 画面が表示されたら成功です。
     左下の「Codespaces」メニュー →「Stop Current Codespace」
     で停止してください。

  ※ うまくいかなくても心配いりません。
    当日、講師がサポートします。
